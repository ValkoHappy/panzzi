
# Тестовое задание: Telegram-бот для анализа публичных телеграм каналов при помощи api LLM.

## Цель

Сделать простого Telegram-бота, который по ссылке на публичный канал собирает посты за последние 3 месяца, считает базовые метрики и возвращает пользователю **один PDF-отчет** со статистикой канала и текстовым аналитическим резюме от LLM.

> Нужна **только одна функция**: пользователь прислал ссылку → получил PDF

---

## Функциональные требования

### 1) Взаимодействие в чате

* Пользователь после старта пишет боту ссылку на телеграм канал.
* На выходе от бота — **один PDF-файл**.

### 2) Сбор данных канала

* Источник — публичный ТГ-канал.
* Период анализа: **последние 3 месяца** 
* Сохраняем по каждому посту как минимум:

  * дата,
  * текст поста,
  * публичная ссылка на пост,
  * просмотры поста,
  * реакции на посте (суммарно все реакции),
  * комментарии (общее количество на посте).

### 3) Метрики в отчете.

PDF отчет должен содержать такие разделы и пункты:

0. Название анализируемого телеграм канала в заголовке отчета.
1. **Подписчики:** - количество подписчиков на канале
2. **Средние просмотры за последние 3 месяца** - под средним я имею ввиду отношение суммы просмотров всех постов на их количество
3. **Средние реакции за последние 3 месяца** 
4. **Средние комментарии за последние 3 месяца**
5. **Активность за последние 3 месяца** - это соотношение средних просмотров к средним реакциям
6. **Самый просматриваемый пост** - нужно указать количество просмотров и ссылку на этот пост, чтобы можно было на него перейти.
7. **Самый комментируемый пост** - нужно указать количество комментариев и ссылку на этот пост, чтобы можно было на него перейти.
8. **Пост с наибольшим количеством реакций** - аналогично
9. **Аналитическое резюме LLM** (текстовый блок) — см. подробнее ниже.

### 4) Нейроаналитика (LLM). Можете использовать api любой нейронки - на ваше усмотрение, которая будет хорошо справляться с этой задачей.

* Соберите тексты постов с телеграм канала за последние 3 месяца.
* Отправьте этот промпт в вашу LLM вместе с текстами постов:

```
Прочитай текст постов Telegram-канала (корпус постов ниже) и сделай краткое аналитическое резюме в 4 разделах:

1) Портрет аудитории — кто читатели, их ценности, стиль речи, уровень формальности и подобное (2–5 тезисов).
2) Сегменты аудитории телеграм канала — 3–5 сегментов с короткими описаниями.
3) Тематическая карта и поведение — ключевые темы/мотивы, поведение аудитории (3–6 пунктов).
4) Рекомендации по формату рекламы — какие форматы и сообщения лучше «зайдут» в этом канале (3–6 пунктов).

Требования к оформлению:
- Чёткие заголовки разделов на русском языке.
- Тезисы короткие, без «воды».


Корпус постов:
<<< ТЕКСТЫ ПОСТОВ >>>
```

* Полученный ответ вставьте в PDF отдельным разделом «**Аналитическое резюме LLM (на основе текстов постов)**».

Пример итогового PDF отчета - скинул отдельно.
---


## Критерии оценки

* Соответствие структуры PDF требованиям.
* Корректность тг парсинга и агрегатов за 3 месяца.
* Обработка ошибок и UX в чате.
* Стабильность работы.
